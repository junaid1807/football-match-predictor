# ⚽ Football Match Predictor

A Machine Learning-based web app that predicts football match outcomes using historical data and team form.

## 🚀 Features
- Match outcome prediction (Win/Draw/Loss)
- Confidence scores with visualization
- Last 5 matches form analysis
- Clean Streamlit UI

## 🧠 Tech Stack
- Python
- Scikit-learn
- Streamlit
- Pandas

## ⚠️ Disclaimer
This project is built for educational purposes only.
It uses static historical data and does not reflect real-time match updates.
